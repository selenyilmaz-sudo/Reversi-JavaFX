# Reversi (Othello) - JavaFX

## Kurulum ve Çalıştırma (Installation Procedure)
1. Projeyi bilgisayarınıza indirin (ZIP olarak veya `git clone` ile).
2. Bilgisayarınızda Java 15 (veya üzeri) ve Maven yüklü olduğundan emin olun.
3. Proje klasörünü tercih ettiğiniz bir IDE (Eclipse, IntelliJ vb.) ile açın.
4. IDE üzerinden projeye sağ tıklayıp **Maven -> Update Project** işlemini gerçekleştirin. Bu işlem `pom.xml` içerisindeki JavaFX ve Media bağımlılıklarını indirecektir.
5. Terminal veya komut satırını açarak proje dizininde `mvn clean javafx:run` komutunu çalıştırın. Oyun otomatik olarak başlayacaktır.
